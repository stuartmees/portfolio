Hidden Gem is a fullstack web app that enables users to register, log-in and post details of 'hidden gems' they found and visited in India. The API was made with a Flask app and a PostgreSQL database and it was to be consumed by a React front end. Google Places API and React Mapbox GL helped users enter location information and display it on the map. I focussed on simple UI design and and clean code with reliable error handling.
